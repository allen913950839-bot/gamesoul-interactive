## 🚀 快速开始（开发者）

### 1. 克隆项目
```bash
git clone <repository-url>
cd GameSoul-Interactive
```

### 2. 安装依赖
```bash
npm install
```

### 3. 配置环境（可选）
```bash
# 复制环境变量模板
cp .env.example .env

# 编辑 .env 文件
# VITE_GEMINI_API_KEY=your_key_here (可选)
# DEEPSEEK_API_KEY=your_key_here (可选)

# 注意: 不配置也能运行，会使用模拟模式
```

### 4. 启动开发
```bash
npm run dev
# 访问 http://localhost:5173
```

### 5. 构建部署
```bash
npm run build
git add .
git commit -m "feat: 新功能"
git push origin main
# Vercel自动部署
```

## 🔧 常见问题

### Q1: 不配置API能用吗？
**A**: 能！会自动使用模拟模式，30+场景化回复，体验完整。

### Q2: 如何添加新角色？
**A**: 修改 `src/App.jsx` 中的 `COMPANIES` 数组，添加角色配置。

### Q3: 如何部署到生产环境？
**A**: `git push` 即可，Vercel自动部署，1-2分钟完成。

### Q4: 数据存在哪里？
**A**: Vercel KV（Redis），持久化存储，免费套餐。

### Q5: 支持多语言吗？
**A**: 当前仅中文，多语言在规划中。
