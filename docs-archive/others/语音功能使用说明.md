# 语音转文字功能使用说明

## 功能概览
✅ **已实现**：录音转文字（语音输入）  
📅 **更新时间**：2025年11月20日

## 使用方法

### 1. 开始语音输入
1. 点击输入框左侧的 **麦克风按钮** 🎤
2. 浏览器会请求麦克风权限，点击"允许"
3. 按钮变为**红色并闪烁**，显示"正在录音中...请说话"
4. 对着麦克风说出你的消息

### 2. 完成输入
- **自动停止**：说完后会自动识别并停止
- **手动停止**：再次点击麦克风按钮
- 识别的文字会自动填入输入框
- 点击发送按钮即可发送消息

## 技术细节

### 支持的浏览器
✅ Chrome / Edge（推荐）  
✅ Safari（iOS/macOS）  
⚠️ Firefox（部分支持）  
❌ 不支持的浏览器会显示提示

### 语音识别引擎
- 使用浏览器内置的 **Web Speech API**
- 语言设置：**中文（zh-CN）**
- 无需额外安装，完全免费

### 权限要求
首次使用时需要授权麦克风权限：
1. Chrome：点击地址栏左侧的锁头图标
2. 找到"麦克风"权限
3. 设置为"允许"

## 界面提示

### 录音状态
| 状态 | 按钮颜色 | 提示文字 |
|------|---------|---------|
| 待机 | 灰色 | - |
| 录音中 | 红色闪烁 | 🎤 正在录音中...请说话 |

### 错误处理
- **没有检测到语音**：请重新点击并大声说话
- **权限被拒绝**：请在浏览器设置中允许麦克风权限
- **浏览器不支持**：请使用Chrome或Edge浏览器

## 常见问题

### Q: 为什么识别不出来？
A: 请确保：
1. 麦克风权限已授权
2. 周围环境不要太吵
3. 说话清晰，语速适中
4. 使用支持的浏览器

### Q: 可以识别方言吗？
A: 目前仅支持**普通话**，方言识别准确度较低

### Q: 能同时录音和打字吗？
A: 可以！语音识别后的文字会填入输入框，可以手动修改

### Q: 有录音时长限制吗？
A: 建议每次录音**不超过30秒**，说完后会自动停止

## 使用技巧

### 💡 最佳实践
1. **安静环境**：降低背景噪音
2. **清晰发音**：语速适中，吐字清楚
3. **短句输入**：建议一次说一句话（10-20秒）
4. **检查修改**：识别后检查文字，必要时手动修正

### 🎯 示例场景
```
用户："今天玩了几把王者荣耀，遇到了很坑的队友"
识别结果：今天玩了几把王者荣耀，遇到了很坑的队友 ✅

用户："我最近在用亚瑟，感觉很强"  
识别结果：我最近在用亚瑟，感觉很强 ✅
```
