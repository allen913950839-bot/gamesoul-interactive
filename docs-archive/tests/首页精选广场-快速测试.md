# 首页精选广场 - 快速测试指南

## 🚀 30秒快速验证

### 1. 启动项目
```bash
npm run dev
```

### 2. 访问首页
打开浏览器：http://localhost:5173

### 3. 验证点
✅ **滚动到底部**  
   → 看到"精选广场"标题 + 🌐 图标

✅ **查看卡片**  
   → 显示3个对话卡片（如果有数据）

✅ **悬停测试**  
   → 鼠标悬停在卡片上  
   → 卡片放大 + 边框变蓝 + 底部显示提示

✅ **点击卡片**  
   → 跳转到广场页

✅ **点击"查看全部"**  
   → 跳转到广场页

## 🐛 常见问题

### Q1: 首页看不到精选广场？
**A**: 检查是否滚动到页面底部，精选广场在游戏选择区域下方

### Q2: 显示"暂无精选内容"？
**A**: 正常现象！需要先创建一些对话并分享到广场

### Q3: 点击卡片没反应？
**A**: 检查控制台是否有报错，确保广场页面路由正常

### Q4: 悬停没有动画效果？
**A**: 检查浏览器是否支持CSS transform，或者禁用了动画

## 📊 测试数据准备

如果广场为空，需要先创建测试数据：

### 方法1：手动创建
1. 选择一个游戏（如王者荣耀）
2. 和亚瑟聊天8-10轮
3. 点击保存按钮
4. 勾选"分享到广场"
5. 填写标题并保存
6. 返回首页刷新

### 方法2：使用Mock数据（开发环境）
在 `api/get-public-conversations.js` 中添加：
