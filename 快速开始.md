# 🚀 GameSoul - 快速开始

## 📦 安装依赖

```bash
npm install
```

## ⚙️ 配置API（可选）

创建 `.env` 文件：

```bash
cp .env.example .env
```

编辑 `.env`，添加API密钥：

```env
# Google Gemini API（亚瑟对话）
VITE_GEMINI_API_KEY=your_gemini_key

# DeepSeek API（光子鸡对话）
DEEPSEEK_API_KEY=your_deepseek_key
```

**获取API Key：**
- **Gemini**: https://makersuite.google.com/app/apikey
- **DeepSeek**: https://platform.deepseek.com

**注意：** 不配置API也能运行，会使用智能模拟回复。

## 🎮 启动项目

```bash
npm run dev
```

访问：http://localhost:5173

## 🎯 核心功能测试

### 1. AI对话
- 点击"王者荣耀"卡片
- 输入任意内容，体验亚瑟的毒舌回复
- 点击"和平精英"卡片，体验光子鸡的萌系对话

### 2. 彩蛋系统
输入特殊符号触发隐藏彩蛋：

| 输入 | 效果 |
|------|------|
| 🔨 | 战锤震击 |
| ⚔️ | 圣剑斩击 |
| 🛡️ | 盾牌防御 |
| 🧪 | 中毒效果 |
| 💎 | 宝石收集 |
| 👑 | 王者加冕 |

每个彩蛋连续使用会有不同强度的表现！

### 3. 语音输入
- 点击麦克风图标
- 允许浏览器访问麦克风
- 开始说话，自动转文字

### 4. 保存与分享
- 点击保存按钮
- 选择"公开分享"或"私密保存"
- 添加标题和标签
- 分享到广场让其他人看到

### 5. 广场功能
- 点击底部"精选广场"
- 浏览其他用户的精彩对话
- 点赞、评论、Remix继续聊天

## 🚀 部署上线

### 方法1：GitHub + Vercel（推荐）

1. **推送到GitHub**：
```bash
git add .
git commit -m "Initial commit"
git push origin main
```

2. **连接Vercel**：
   - 访问 https://vercel.com
   - 使用GitHub登录
   - Import项目
   - 添加环境变量（同.env内容）
   - 点击Deploy

3. **获取在线链接**：
   - 部署完成后获得类似：`https://your-project.vercel.app`
   - 每次git push会自动重新部署

### 方法2：快捷脚本

使用项目内的自动化脚本：

```bash
./update.sh
```

## 🐛 常见问题

**Q: AI回复都是固定的？**  
A: 检查 `.env` 中的API Key是否正确配置。未配置会使用模拟模式。

**Q: 语音识别不工作？**  
A: 确保使用Chrome/Edge浏览器，并授予麦克风权限。

**Q: 保存功能失败？**  
A: 需要配置Vercel KV数据库，参考《部署指南.md》。

**Q: 彩蛋没反应？**  
A: 确保输入的是Emoji符号（🔨⚔️🛡️等），不是文字。

## 📊 数据库配置（可选）

保存功能需要Vercel KV数据库：

1. 在Vercel项目中创建KV数据库
2. 添加环境变量：
```env
KV_REST_API_URL=your_kv_url
KV_REST_API_TOKEN=your_kv_token
```

详细步骤参考《部署指南.md》的数据库配置章节。
