## 🤔 常见问答

### Q1: 精选对话是如何选出来的？
**A**: 
```
目前按热度排序，综合考虑：
- 浏览量（40%权重）
- 点赞数（40%权重）
- 新鲜度（20%权重）

未来会加入：
- 智能推荐算法
- 个性化推荐
- 编辑精选
```

---

### Q2: 只展示3条会不会太少？
**A**: 
```
经过设计考虑：
- 首页主功能是游戏选择，广场是辅助
- 3条恰好够吸引注意，不会喧宾夺主
- 想看更多有"查看全部"按钮
- 移动端3条刚好一屏，体验最佳

数据会告诉我们是否需要调整。
```

---

### Q3: 如果没有对话怎么办？
**A**: 
```
有友好的空状态提示：
- 显示"暂无精选内容"
- 不影响主流程（仍可选游戏）
- 引导用户创建对话

初期可以：
- 官方创建示例对话
- 内测用户提前铺垫内容
```

---

### Q4: 会不会影响首页加载速度？
**A**: 
```
性能优化措施：
- 只加载6条数据（轻量）
- 异步加载（不阻塞主流程）
- 失败降级（出错不影响使用）
- 构建时间 < 1s（实测验证）

实际体验：几乎无感知。
```

---

### Q5: 手机上效果如何？
**A**: 
```
完全响应式设计：
- 移动端单列布局
- 卡片自适应宽度
- 触摸交互优化
- 动画性能优化

建议现场演示手机效果。
```
